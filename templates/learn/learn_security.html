If the credit card details are sent in the clear text from the client to the server, an attacker is able to sniff the communication channel, and read the credit card details.

In conclusion, with security, sensitive information can be secured using encryption. This stops attackers from being able to read the data that is sent over, thus protecting sensitive information to be seen by unauthorised people.
