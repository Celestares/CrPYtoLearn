Trusted Systems and Reference Monitor

What is a trusted system?
  • A computer system that can be trusted to a specified extent
  • Enforces a specified security policy
  • Implemented using a reference monitor
What is a reference monitor?
  • An entity at the heart of the computer system
  • Responsible for all decisions related to access controls

Taken from Applied Cryptography lecture notes

In the above picture, if the subject request for an action on an object (e.g. reading a file), The reference monitor will check the request to determine if the subject has the privilege to do so. If he/she has, access is granted. If not, access is denied.

3 characteristics a reference monitor have:
  1. Should be tamperproof
  2. Should always be invoked (Enable enforcement of all requests)
  3. Should be small enough (Enable itself to be independently tested)
